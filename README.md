# Simba

Simba is a generator for Sinatra applications.

Simba chooses slim + sinatra + sequel to build your awesome apps.

Simba's aim to set up ruby off rails's best practices.

## Installation

    $ gem install simba

## Usage

    $ simba appname [--skip-bundle]
